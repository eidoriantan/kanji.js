Kanji character object
What a Kanji character object looks like
{% capture character %} ```javascript // Kanji character object { freq: 1, grade: 1, jlpt: 4, kunyomi: ['ひ', '-び', '-か'], literal: '日', meanings: ['day', 'sun', 'Japan', 'counter for days'], onyomi: ['ニチ', 'ジツ'], stroke_count: 4 } ``` {% endcapture %} {{ character | markdownify }}
Methods
Static Method: getDetails(character)
Return type: object

Gets the kanji details of the character and returns it. If the character was not found on the dictionary, this returns null.

Static Method: search(options = {})
Return type: object[]

Searches the entire dictionary file for words that matches the options.

{% capture search-options %} ```javascript // Available options { jlpt: 1, grade: 1, romaji: 'hi', meaning: 'counter for days' } ``` {% endcapture %} {{ search-options | markdownify }}
Static Method: random()
Return type: object

Returns a random kanji character.
