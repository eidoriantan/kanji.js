kanji.js

kanji.js is an open sourced JavaScript library used to search/lookup 漢字 (kanji) characters.

{% octicon browser height:72 aria-hidden:true %}

Node & Browser

It can run on both Node.js and browsers.

{% octicon law height:72 aria-hidden:true %}

Lightweight

It only weights about ~2MB including the dictionary file!
